# 釐清問題

關於「課程結構 (Curriculum Structure)」，章節 (Chapter) 與 單元 (Lesson) 目前在 ERM 中缺乏「排序 (Order/Sequence)」欄位。請問是否需要在資料庫中明確儲存排序資訊，還是依賴建立時間或標題？

# 定位

ERM: Table Chapter, Table Lesson

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 新增排序欄位：在 Chapter 與 Lesson 新增 `sequence_number` (int) 以明確控制順序 |
| B | 依建立時間：預設依照 `created_at` 排序 (不建議，難以調整) |
| C | 依標題：依照 `title` 字母順序 (不實用) |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- ERM (Chapter, Lesson 欄位)
- API Response (List ordering)

# 優先級

High
