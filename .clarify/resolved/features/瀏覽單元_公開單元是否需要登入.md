# 釐清問題

瀏覽公開單元 (Public Lesson) 時，使用者是否需要先登入系統？

# 定位

Feature：瀏覽單元.feature
Rule：若單元設為公開，則無需購買即可觀看

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 是，必須登入才能觀看（即使無需購買） |
| B | 否，訪客（未登入）也可觀看 |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- 瀏覽單元.feature (Pre-condition: Given 使用者已登入?)
- Auth Middleware 設定
