# 釐清問題

若課程狀態變更為「未發布 (Unpublished)」，已經購買該課程的使用者是否仍可瀏覽與觀看內容？

# 定位

Feature：瀏覽課程.feature
Rule：未發布的課程對一般使用者隱藏

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 是，已購買者仍可存取（權益保障） |
| B | 否，強制隱藏，無法存取（如課程下架） |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- 瀏覽課程.feature
- 瀏覽單元.feature (Access Control Rule)

# 優先級

High

---
# 解決記錄

- **回答**：B - 否，強制隱藏，無法存取（如課程下架）
- **更新的規格檔**：spec/features/瀏覽課程.feature
- **變更內容**：在「未發布的課程對一般使用者隱藏」規則中，加入「即使已購買也無法存取」的說明與 Example，明確規範未發布課程對所有使用者（含已購者）皆不可見。
